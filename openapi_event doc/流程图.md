# 设备报警消息
<!-- 这是一个文本绘图，源码为：flowchart TD
    Device[设备]

    Device -->|HTTP协议（报警消息）| AlarmService[报警服务<br/>（检查开发者账号）]
    Device -->|MQTT协议（报警消息）| MQTTBroker[MQTT Broker]
    MQTTBroker -->|订阅推送（报警消息）| AlarmService

    AlarmService -->|检查通过后，AMQP协议（报警消息）| RabbitMQ[RabbitMQ]
    RabbitMQ -->|推送报警消息| PushService[消息推送服务]

    PushService -->|调用第三方云接口（报警消息）| ThirdPartyCloud[第三方云] -->
